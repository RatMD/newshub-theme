NewsHub uses Bootstrap v5.2 with the awesome Bootstrap Icons, which can be implemented into any CMS Page using the following partial syntax:

Next to the Bootstrap native Carousel, NewsHub also implements the responsive Vanilla JavaScript Slider Keen-Slider, including a simple HTML data-attribute initialization:

<div class="keen-slider" data-keen-slider='{ ... }'>

Light Mode

The NewsHub OctoberCMS Template is available in two color schemes: Light and Dark. You can either decide which color scheme should be used, or you enable the color switch button (on the header or footer) and let your visitors decide!
