NEWSHUB

NewsHub is a minimalistic Premium OctoberCMS (v2 & v3) template, especially designed to build up a rich and responsive Blog or Magazine website in no time. The whole package is pre-configured out of the box, thus you don't need to set-up many things before start writing and publishing your articles! However, when you're looking for some extra flavour, to make your website even more unique, NewsHub provides you with a ready-to-use node.js environment using famous and familiar tools, such as Rollup and Bootstrap.

Prominent Features

Bootstrap v5.2

NewsHub uses Bootstrap v5.2 with the awesome Bootstrap Icons, which can be implemented into any CMS Page using the following partial syntax:

Light Mode

The NewsHub OctoberCMS Template is available in two color schemes: Light and Dark. You can either decide which color scheme should be used, or you enable the color switch button (on the header or footer) and let your visitors decide!
